# Ann Exhibition
> **서울 숲 갤러리아포레 <내 이름은 빨강머리 앤>전시 가이드 및 웹 서비스 시스템**

> 본 프로젝트는 전시의 시각적 요소를 디지털 환경으로 확장하기 위해 기획되었습니다.
> SCSS를 활용한 체계적인 스타일링과 Supabase를 통한 실시간 데이터 연동을 통해 전시정보와 프로그램을 직관적으로 제공하는 웹서비스를 구현하였고 Vercel를 활용해 최적화된  배포 환경을 구축했습니다.

🔗 Deploy
https://jeju-portfolio.netlify.app/

## 🛠 기술 스택 및 개발 환경
### **Frontend**
- React (Vite)
- TypeScript
- SCSS (Mixins 기반 디자인 시스템)

### **Backend & Database**
- Supabase

### Deployment & Tool
- Vercel
- Figma / Github

## ✨ 주요 기능 및 기술 구현 포인트
### Supabase 실시간 데이터 연동
- 실제 DB에 저장된 작품 정보, 프로그램 데이터 등을 API로 호출하여 화면에 동적으로 출력

### SCSS 기반의 디자인 시스템
- 전시회 대표컬러나 포인트 컬러, 폰트설정들을 Mixin을 통해 중복 코드를 최소화화고 유지보수에 용이한 스타일 설계

### 반응형 UI/UX
- 구현중
